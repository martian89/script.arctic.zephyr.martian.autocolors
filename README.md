## Automatic Day and Night Colors for Arctic: Zephyr - Martian [![License](https://img.shields.io/badge/License-GPLv3-blue)](https://www.gnu.org/licenses/gpl-3.0.txt)

This add-on switches the skin color theme based on time

The time can be set manually or you can set your location and the theme will be switched based on sunset/sunrise.

To disable the automatic switching either:
- Deactivate this add-on
- Go to "Skin Settings - Furniture" and deactivate the automatic switching
